# prompt-engineering-expert — Craft, optimize, and debug prompts like a pro

> An advanced prompt engineering assistant for writing, refining, and testing AI prompts and system instructions. Applies proven techniques and anti-pattern recognition to make prompts precise, consistent, and reliable.

[![OpenClaw Skill](https://img.shields.io/badge/OpenClaw-Skill-blueviolet)](https://github.com/NachaFromMars)

## Overview
prompt-engineering-expert covers the full prompt lifecycle — from writing a clear first draft to systematic optimization, evaluation with test cases, and anti-pattern debugging. It applies advanced techniques (chain-of-thought, few-shot examples, XML tags, role-based prompting) and handles context/token management and multimodal prompting (vision, embeddings, file-based).

## Features
- **Prompt writing best practices** — clear, direct, structured
- **Custom instructions design** — system prompts for AI agents and skills
- **Prompt optimization** — analyze, refine, improve existing prompts
- **Advanced techniques** — chain-of-thought, few-shot, XML tags, role-based prompting
- **Evaluation & testing** — test cases + success criteria
- **Anti-pattern recognition** — spot and fix common mistakes
- **Context management** — token optimization for long contexts
- **Multimodal prompting** — vision, embeddings, file-based inputs

## Usage / Quick Start
Bring a prompt or a goal. Examples:
- *"Optimize this system prompt for a customer support agent"*
- *"Add few-shot examples to make this consistent"*
- *"Why does this prompt keep giving vague answers?"*
